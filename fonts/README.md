# Fonts

Drop these 5 BCG Henderson `.woff2` files here to activate the full Henderson type system in `index.html`:

- `BCGHendersonSans-Thin.woff2`     (weight 100)
- `BCGHendersonSans-Light.woff2`    (weight 300)
- `BCGHendersonSans-Regular.woff2`  (weight 400)
- `BCGHendersonSans-Bold.woff2`     (weight 700)
- `BCGHendersonSerif-Regular.woff2` (weight 400, serif)

Until the files land, the toolkit falls back to **Trebuchet MS** (Windows/Mac system font) → Inter → system-ui. Trebuchet gives the BCG look-and-feel immediately, so missing Henderson files won't break anything.

Source: exported from the BCG `bcg-u-brand-html` Claude.ai skill (or the `deckster-slide-generator` skill's `styles/` or `assets/` folder).
